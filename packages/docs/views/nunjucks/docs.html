ATX Documentation

Installation

npm install atx

Configuration

Create an app.config.js file in your project root:

module.exports = {
  scripts: './scripts/main.js',  // Entry point for JavaScript
  styles: './styles/main.scss',  // Entry point for Sass
  views: './views',             // Templates directory
  assets: './assets',          // Static assets
  outputDir: './dist',        // Build output
  data: {                    // Global data
    title: 'My Site'
  },
  compiler: (template, data) => {
    // Your template compiler
  }
}

Commands

Build

npx atx build

Development

npx atx watch

Template Engines

ATX supports various template engines. Here are some examples:

Nunjucks

const nunjucks = require('nunjucks');

module.exports = {
  compiler: (template, data) => {
    return nunjucks.renderString(template, data);
  }
}

Handlebars

const Handlebars = require('handlebars');

module.exports = {
  compiler: (template, data) => {
    const compiled = Handlebars.compile(template);
    return compiled(data);
  }
}
